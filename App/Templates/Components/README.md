# Template Components

Reusable building blocks shared across pages and modules.

Current components:

- `BadgeList.vide`
- `CodeList.vide`
- `DataTable.vide`
- `DefinitionGrid.vide`
- `LinkList.vide`
- `ProductGrid.vide`

`.php` and `.lmv` counterparts are retained as compatibility fallbacks; new first-party edits should start with `.vide`.
